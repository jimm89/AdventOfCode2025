## Part 1

### Understanding the problem

This is a classic 'modular artihmetic' problem. What this means is that we only use the numbers 0 to MOD - 1 inclusive, and when we get to a certain number (let's call it MOD), we go back to 0, and when we go below zero, we start counting down again from MOD - 1.
